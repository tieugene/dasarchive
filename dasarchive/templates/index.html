DasArchive

Описание
Система управления файлопомойкой
Назначение:
Долговременное хранилище большого количества важных файлов
А именно:
+ быстро залить
- медленно рассортировать
+ быстро найти
+ быстро открыть
+ быстро записать

Порядок действий:

  1. Заходим файлменеджером в WebDAV-ресурс для входящих файлов: KDE Dolphin , GNOME Nautilus , Windows (Сетевое окружение - Добавить новое место - вставить эту ссылку - Ok, Ok)
  2. Мышкой оптом копируем/переносим туда файлы со своего компьютера.
  3. Идем в Inbox - здесь видны наши залитые файлы. Добавляем их в хранилище, расставляя Теги .
  4. Идем в Файлы - это файлы уже в хранилище. Фильтруем, редактируем, удаляем. У каждого файла будут ссылки для прямого доступа по протоколу WebDAV.
Примечания:
  • Linux:
    • С Kоnqueror файлы надо открывать приложениями KDE, с Epiphany - приложениями GNOME
    • Epiphany работает нормально, но надо хотя бы один раз открыть dav-ссылку (папка любого файла) в Nautilus
    • FireFox можно научить перенаправлять webdav:// на dolphin или dav:// на nautilus (только папки файлов):
      • about:config
      • "Я обещаю"
      • Правая педаль - Создать - Логическое - network.protocol-handler.expose.webdav - false
      • При первом же выборе ссылки webdav://... выбрать /usr/bin/dolphin (или konqueror)
      • Аналогично - dav и nautilus
  • Windows (доступ к файлам):
    • Добавить эту ссылку в Сетевое окружение (Сетвое окружение - новое место в сетевом окружении - ).
    • Уговорить браузер открывать ссылки file://.
    • Opera и Chrome не открывают, IE8 и FireFox - работают.
