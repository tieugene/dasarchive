1. Все Tag файла (cTagss):
	?	START f=node(file_id) MATCH (t)->(f) RETURN t;
2. Facet выбранных Tag (cFacets):
	+	START t=node(tag_id[s]) MATCH (f)-[:has_tag*]->(t) WHERE f.element_type = "facet" RETURN f;
3. Все Tag на пути Facet=>Tag (iTags):
		START f=node(facet_id[s]), t=node(tag_id[s]) MATCH (f)-[:has_tag*]-(i)->(t) RETURN i;
4. Все Facet этих Tags (eFacets):
		START t=node(tag_id[s]) MATCH (f)-[:has_facet*]->(t) RETURN f;
5. Все Tags этих Facets (eTags):
		START f=node(facet_id[s]) MATCH (f)-[:has_tag*]->(t) WHERE t.element_type = "tag" RETURN t;

So:
- choosen tags (tag(*)->file = cTags)
- choosen facets (allShortestPaths(root->file)|facet = cFacets)
- intermediate tags (allShortestPaths(root->file)|tag / aSP(root->cTags)|tag = iTags)
- choosen edges (cTags|iTags|cFacets<>same)
- enabled facets
- enabled tags
- enabled edges
- disabled facets
- disabled tags
- disabled edges
