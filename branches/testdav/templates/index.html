DAV demo

Демонстрация совмещения web-ui и прямого доступа к файлам по протоколу WebDAV.

Порядок действий:

  1. Windows:
    • Добавить этот ресурс в сетевое окружение (Сетевое окружение - Добавить новое место - вставить ссылку - Ok, Ok)
    • Уговорить (может и этим ) браузер открывать ссылки file://
  2. Заходим файлменеджером в WebDAV-ресурс: KDE Dolphin , GNOME Nautilus , Windows ® Explorer
  3. Копируем туда файлы со своего компьютера.
  4. Открываем ссылку в Konqueror (KDE), Epifany (GNOME) или Firefox (Microsoft ® Windows ®) соответственно.
  5. Добавляем/изменяем/удаляем "comment".
  6. Открываем файлы по прямой WebDAV-ссылке.
